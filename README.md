# 🤖 HR Employee Support Chatbot

## 📌 Project Title

**HR Employee Support Chatbot (PeopleDesk AI)**

---

## 🎯 Aim

To design and develop an **HR Employee Support Chatbot** using HTML, CSS, and JavaScript that provides instant responses to employee queries.

---

## 📝 Description

The HR chatbot is a **web-based assistant** designed to help employees quickly access HR-related information without contacting HR manually.

It handles:

* Leave policies
* Salary details
* Company policies
* Employee benefits
* Basic conversations (Hi, Bye, Thank you)

---

## 🎯 Objectives

* Reduce HR workload
* Provide instant employee support
* Improve communication efficiency
* Automate repetitive HR queries

---

## ⚙️ Working Principle

The chatbot uses a **rule-based keyword matching system**.

### Process:

1. User enters a query
2. Input is converted to lowercase
3. Keywords are matched
4. Matching category is detected:

   * Leave
   * Salary
   * Policy
   * Benefits
5. Response is generated dynamically
6. Displayed in chat UI

---

## 🧠 Algorithm

1. Start chatbot
2. Initialize dataset
3. Accept user input
4. Normalize text
5. Match keywords
6. Generate response
7. Display output
8. Repeat

---
